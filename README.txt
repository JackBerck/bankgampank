Cara Menjalankan Proyek BankGampank
1. Jalankan xampp atau laragon atau aplikasi local server lainnya lalu mulai jalankan server lokalnya
2. Buat database pada phpMyAdmin dengan nama "bankgampank" lalu import databasenya yang berada pada folder proyek dengan nama file bankgampank.sql
3. Ekstrak file bankgampank.zip
4. Buka NetBeans lalu open project "bankgampank" yang sebelumnya sudah diekstrak
5. Jalankan aplikasinya melalu NetBeans
